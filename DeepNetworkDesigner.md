# GoogLeNet 與 SqueezeNet 於電子產品影像分類之效能比較

本研究旨在探討兩種經典卷積神經網路（CNN）模型：**GoogLeNet** 與 **SqueezeNet**，在特定領域（電子產品）影像辨識上的表現差異。

## 📌 專案摘要
* **目標**：辨識手機、平板、電腦、遊戲機、智慧型手錶五類電子產品。
* **技術棧**：MATLAB, Deep Network Designer, CNN.
* **資料集**：250 張影像

## 📊 實驗結果對照
在測試集上的辨識準確率如下表所示：

| 類別 | SqueezeNet (輕量級) | GoogLeNet (多尺度) |
| :--- | :--- | :--- |
| **智慧型手錶** | 100% | 100% |
| **電腦** | 99.2% | 99.4% |
| **手機** | 98.1% | 95.6% |
| **平板** | 93.0% | 76.2% |
| **遊戲機** | 78.4% | 71.0% |




## 🔍 核心發現
1.  **結構穩定性**：GoogLeNet 在多尺度特徵擷取上表現優異，但由於層數較深，在小型資料集上若未充分訓練，準確率（如平板類）可能不如結構簡單的 SqueezeNet。
2.  **效能權衡**：SqueezeNet 雖然參數極少，但在本專案的分類任務中展現出極高的 CP 值，適合部署於行動裝置。

## 🛠️ 遇到問題與解決
* **格式統一**：使用 Xnview 將所有不同來源的圖片統一轉換為 `.jpg`。
* **模型輸出修正**：解決了訓練資料夾類別數量與模型輸出層（Output Layer）不匹配的問題。
